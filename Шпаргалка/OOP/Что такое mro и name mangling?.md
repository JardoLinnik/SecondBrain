* mro() - метод, который возвращает список классов, от которых пронаследован объект. чем раньше класс в списке, тем выше приоритет. это нужно для решения проблемы ромбовидного наследования, когда у двух родительских классов один и тот же метод определен по разному. дочерний класс берет метод того класса, который в списке идет раньше.  
вид: [<class '__main__.D'>, <class '__main__.B'>, <class '__main__.C'>, <class '__main__.A'>, <class 'object'>]  
  
* name mangling - это механизм изменения имен атрибутов классов с целью предотвращения конфликтов имен в иерархии наследования.  
Выглядит как: obj._ClassName__attribute
