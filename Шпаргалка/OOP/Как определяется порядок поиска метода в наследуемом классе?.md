Если у двух родителей класса есть одноимённые методы, то дочерний класс наследует его у того родителя, который при перечислении его родителей указан раньше. Также метод mro() отображает порядок поиска наследуемого метода.  
  
```  
class A:  
def method(self):  
print("Метод из класса A")  
  
class B(A):  
def method(self):  
print("Метод из класса B")  
  
class C(A):  
def method(self):  
print("Метод из класса C")  
  
class D(B, C):  
pass  
  
# Создание экземпляра класса D  
d = D()  
  
# Вызов метода  
d.method()  
  
# Вывод порядка разрешения методов (MRO)  
print(D.__mro__)  
# (<class '__main__.D'>, <class '__main__.B'>, <class '__main__.C'>, <class '__main__.A'>, <class 'object'>)  
# Наследуется метод из класса B, т.к. он в наследовании и результате mro() указан раньше  
```
