*Итератор* - это объект, который поддерживает протокол итерации, т.е он реализует два дандер метода `__iter__()` и `__next__()`  
  
Эти методы позволяют обходить элементы коллекции по одному, не загружаю всю коллекцию в память  
  
#### Метод `__iter__()`  
* Возвращает сам объект итератора. Вызывается в начале цикла `for` или при явном вызове `iter()`  
* Если объект уже является итератором, то `__iter__()` возвращает `self`  
#### Метод `__next__()`  
* Возвращает следующий элемент последовательности, если элементов больше нет, то выбрасывает исключение `StopIteration`  
  
### Реализация кастомного итератора  
  
```python  
class MyIterator:  
def __init__(self, data):  
self.data = data  
self.index = 0  
def __iter__(self):  
return self  
def __next__(self):  
if self.index >= len(self.data):  
raise StopIteration  
result = self.data[self.index]  
self.index += 1  
return result  
  
### Важные моменты  
* Итераторы полезны для обработки больших объемов данных, т.к не загружают их сразу в память  
* В Python встроены различные итераторы, например, файлы, генераторы, объекты из range, map, filter  
* Все итераторы являются итерируемыми объектами, но не все итерируемые объеты являются итераторами. Например: списки или строки являются итерируемыми, но не являются итераторами, т.к не имеют __next__()
