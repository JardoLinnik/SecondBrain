Метакласс — это класс, который управляет созданием других классов. Для создания метакласса нужно наследоваться от класса type.  
Если классы — это "шаблоны" для создания объектов, то метаклассы — это "шаблоны" для создания классов.  
  
Пример:  
  
```  
class MyMeta(type):  
def __new__(cls, name, bases, dct):  
# Создание нового класса  
print(f"Создаём класс {name}")  
return super().__new__(cls, name, bases, dct)  
  
  
def __init__(cls, name, bases, dct):  
# Инициализация класса  
print(f"Инициализируем класс {name}")  
super().__init__(name, bases, dct)  
  
  
  
  
class MyClass(metaclass=MyMeta):  
pass  
```  
  
name — имя класса.  
bases — кортеж базовых классов, от которых наследуется новый класс.  
dct — словарь атрибутов и методов класса.  
  
Зачем использовать метаклассы:  
  
1. Автоматическое добавление методов: Вы можете автоматически добавлять или изменять методы в классах, не изменяя их вручную.  
2. Валидация классов: Проверка наличия обязательных атрибутов и методов.  
  
Примеры использования:  
* В Django ORM метаклассы валидируют поля модели и добавляют метаинформацию классов полей в класс модели.  
* В Pydantic за счёт метаклассов происходит валидация полей, десериализация json, создание схемы модели в виде json
