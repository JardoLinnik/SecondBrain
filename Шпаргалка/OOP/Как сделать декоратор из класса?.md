Необходимо определить класс с методом __init__ и методом __call__.  
Пример:  
  
```  
class Logger:  
def __init__(self, func):  
self.func = func # Сохраняем декорируемую функцию  
  
  
def __call__(self, *args, **kwargs):  
print(f"Вызов функции '{self.func.__name__}' с аргументами {args} и {kwargs}")  
result = self.func(*args, **kwargs)  
print(f"Функция '{self.func.__name__}' вернула {result}")  
return result  
  
@Logger  
def say_hello():  
pass  
```
