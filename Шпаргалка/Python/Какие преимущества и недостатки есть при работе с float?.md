
Преимущества: скорость вычислений очень быстрая за счет вычислений на процессоре. Недостатки: вычисления во float происходят с потерей точности, поэтому с помощью этого типа нельзя делать расчеты там, где она важна. Для того чтобы достичь точности нужно использовать тип decimal или переводить все в тип int если это возможно.
