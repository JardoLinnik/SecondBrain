Синтаксически это выглядит как функция, находящаяся целиком в теле другой функции. При этом вложенная внутренняя функция содержит ссылки на локальные переменные внешней функции. Каждый раз при выполнении внешней функции происходит создание нового экземпляра внутренней функции, с новыми ссылками на переменные внешней.  
Пример:  
  
```  
def outer():  
x = 10 # Переменная из внешней функции  
  
  
def inner():  
print(x) # Использует переменную из внешней функции  
return inner # Возвращаем внутреннюю функцию  
  
  
closure = outer() # Создаем замыкание  
closure() # Вывод: 10  
```  
  
Замыкания часто используются для создания фабрик функций, функций с состоянием и в качестве альтернативы классам (для инкапсуляции), если нужен доступ к данным, но не нужна полная структура класса. Декоратор - частный случай замыкания.
