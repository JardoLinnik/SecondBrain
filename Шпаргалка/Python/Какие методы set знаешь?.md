* Модифицирующие методы (изменяют исходное множество)  
  
.add(elem)  
Назначение: добавляет элемент в множество. Если элемент уже присутствует, ничего не происходит.  
Возвращает: None.  
  
.remove(elem)  
Назначение: удаляет элемент из множества. Если элемента нет, возбуждается KeyError.  
Возвращает: None.  
  
.discard(elem)  
Назначение: удаляет элемент из множества, если он существует. Если элемента нет, исключения не возникает.  
Возвращает: None.  
  
.pop()  
Назначение: удаляет и возвращает произвольный элемент множества. Если множество пустое, возбуждает KeyError.  
Возвращает: удалённый элемент.  
  
.clear()  
Назначение: удаляет все элементы из множества, делая его пустым.  
Возвращает: None.  
  
.update(*others)  
Назначение: добавляет в текущее множество все элементы из других множеств или итерируемых объектов. Аналог операции |=.  
Возвращает: None.  
  
.intersection_update(*others)  
Назначение: оставляет в множестве только те элементы, которые присутствуют во всех переданных множествах или итерируемых объектах. Аналог операции &=.  
Возвращает: None.  
  
.difference_update(*others)  
Назначение: удаляет из текущего множества все элементы, которые есть в других множествах или итерируемых объектах. Аналог операции -=.  
Возвращает: None.  
  
.symmetric_difference_update(other)  
Назначение: заменяет текущее множество множеством, состоящим из элементов, которые присутствуют только в одном из множеств (исключающее ИЛИ). Аналог операции ^=.  
Возвращает: None.  
  
* Возвращающие новые множества (не изменяют исходное)  
.union(*others)  
Назначение: возвращает новое множество, содержащее объединение текущего множества и всех переданных множеств/итераций. Аналог операции |.  
Возвращает: новое множество.  
  
.intersection(*others)  
Назначение: возвращает новое множество, содержащее пересечение текущего множества и всех переданных множеств/итераций. Аналог операции &.  
Возвращает: новое множество.  
  
.difference(*others)  
Назначение: возвращает новое множество, содержащее элементы текущего множества, отсутствующие в других множествах/итерациях. Аналог операции -.  
Возвращает: новое множество.  
  
.symmetric_difference(other)  
Назначение: возвращает новое множество, содержащее элементы, которые присутствуют только в одном из множеств (исключающее ИЛИ). Аналог операции ^.  
Возвращает: новое множество.  
  
* Проверочные методы (возвращают булево значение)  
.issubset(other)  
Назначение: проверяет, является ли текущее множество подмножеством other. Аналог <=.  
Возвращает: True или False.  
  
.issuperset(other)  
Назначение: проверяет, является ли текущее множество надмножеством other. Аналог >=.  
Возвращает: True или False.  
  
.isdisjoint(other)  
Назначение: проверяет, не имеют ли множества общих элементов.  
Возвращает: True или False.  
  
* Копирование  
.copy()  
Назначение: возвращает поверхностную копию текущего множества.  
Возвращает: новое множество (shallow copy).
