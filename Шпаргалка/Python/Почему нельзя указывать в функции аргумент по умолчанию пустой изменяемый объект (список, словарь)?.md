Потому что объект будет создан при инициализации функции, из-за чего он будет накапливать в себе значения после каждого вызова функции, что приведет к неожиданному поведению.  
  
Пример:  
  
```  
def add_item_to_list(item, items=[]):  
items.append(item)  
return items  
  
print(add_item_to_list("apple")) # ['apple']  
print(add_item_to_list("banana")) # ['apple', 'banana']  
print(add_item_to_list("melon")) # ['apple', 'banana', 'melon']  
```
