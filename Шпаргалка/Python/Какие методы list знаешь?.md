.append(x)  
Назначение: добавляет элемент x в конец списка.  
Возвращает: None.  
  
.extend(iterable)  
Назначение: добавляет все элементы из iterable (список, кортеж, любая итерируемая структура) в конец текущего списка.  
Возвращает: None.  
  
.insert(i, x)  
Назначение: вставляет элемент x перед индексом i (смещая текущий элемент на этом индексе и остальные вправо).  
Возвращает: None.  
  
.remove(x)  
Назначение: удаляет первое вхождение элемента x в списке (если такого элемента нет, возбуждается ValueError).  
Возвращает: None.  
  
.pop(i=-1)  
Назначение: удаляет и возвращает элемент списка по индексу i. По умолчанию (i=-1) удаляет и возвращает последний элемент. Если список пуст, вызовется исключение IndexError.  
Возвращает: элемент, который был удалён.  
  
.clear()  
Назначение: удаляет все элементы из списка, оставляя пустой список.  
Возвращает: None.  
  
.index(x, start=0, end=None)  
Назначение: возвращает индекс первого вхождения элемента x в списке; поиск идёт от start до end (если они указаны).  
Возвращает: целое число (индекс).  
  
.count(x)  
  
Назначение: считает, сколько раз элемент x встречается в списке.  
Возвращает: целое число (количество вхождений).  
  
.sort(key=None, reverse=False)  
Назначение: сортирует список на месте (in-place) через TimSort.  
Возвращает: None.  
Примечание: можно передать key — функцию (или лямбду), которая будет использоваться для сравнения, а также reverse=True для обратной сортировки.  
  
Назначение: разворачивает список на месте.  
Возвращает: None.  
  
.copy() (добавлен в Python 3.3+)  
  
Назначение: возвращает поверхностную копию списка.  
Возвращает: новый список.
