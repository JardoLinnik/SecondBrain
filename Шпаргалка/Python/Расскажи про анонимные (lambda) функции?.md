Lambda-функции в Python — это анонимные (безымянные) функции, которые можно объявить «на лету». Они создаются с помощью ключевого слова lambda и могут содержать только одно выражение (результат этого выражения возвращается автоматически).  
  
Основные моменты:  
* Однострочность — не могут использовать if else (могут только в виде тернарного оператора), циклы, операторы (return), инструкции (pass, raise, continue, try, except), документацию, иначе будет выброшен SyntaxError  
* Анонимность — у функции нет имени (хотя можно присвоить её переменной, если нужно).  
* Частый кейс использования — в качестве «быстрых» колбэков в функциях высшего порядка (map, filter, sorted, и т.п.).  
  
Пример использования:  
  
```  
# Обычная функция:  
def add(a, b):  
return a + b  
  
# Lambda-функция:  
add_lambda = lambda a, b: a + b  
  
print(add(2, 3)) # 5  
print(add_lambda(2, 3)) # 5  
```  
  
```  
nums = [1, 2, 3, 4, 5]  
squared = list(map(lambda x: x**2, nums))  
print(squared) # [1, 4, 9, 16, 25]  
```  
Таким образом, lambda-функции удобны, когда нужно кратко объявить простую функцию «по месту», не давая ей имени и не вынося отдельное определение.
