Как происходит интерпретация в Python?  
  
1. Когда вы запускаете Python-скрипт, интерпретатор читает исходный код.  
2. Преобразует его в байт-код (файлы с расширением .pyc).  
3. Виртуальная машина Python (PVM) исполняет этот байт-код строчка за строчкой.  
  
Чем байт-код отличается от машинного кода?  
  
* Байт-код — это промежуточное представление, понятное только виртуальной машине Python;  
- он не зависит напрямую от аппаратной архитектуры.  
- состоит из инструкций (например, LOAD_CONST, CALL_FUNCTION, RETURN_VALUE и т. п.), названий переменных и чисел(индексов переменных и инструкций). Можно увидеть в человеко-читаемом виде с помощью модуля dis.  
- Хранится в .pyc файлах в шестнадцатиричном виде. Сами файлы хранятся в директории __pycache__  
  
Пример Байт-кода в человеко-читаемом виде:  
  
Код Python:  
  
```  
def add(a, b):  
return a + b  
  
import dis  
dis.dis(add)  
```  
  
Байт-код:  
  
```  
2 0 LOAD_FAST 0 (a)  
2 LOAD_FAST 1 (b)  
4 BINARY_ADD  
6 RETURN_VALUE  
```  
  
* Машинный код (ELF на Linux, Mach-O на macOS, PE на Windows) — это низкоуровневые инструкции для конкретного процессора и операционной системы. Представляет собой последовательность байт (нулей и единиц).
