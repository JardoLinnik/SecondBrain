* Python 3.7:  
- Датаклассы  
- Можно добавлять несколько контекстных менеджеров в один блок with  
- Словари сохраняют порядок вставки ключей  
* Python 3.8:  
- Моржовый оператор :=  
* Python 3.9:  
- Операторы слияния и обновления словарей (| и |=)  
- Синаксис list[int] при типизации, не нужно импортировать List из typing  
- Модуль graphlib для работы с графами  
* Python 3.10:  
- match case (аналог switch case)  
- | вместо Union в типизации  
- Улучшение сообщений об ошибках  
* Python 3.11:  
- Ускорение интерпретатора  
- Можно использовать несколько except в одном блоке  
- Модуль tomlib для чтения .toml  
- Добавлены группы искдючений (ExceptionGroup)  
* Python 3.12:  
- Суб Интерпретаторы  
* Python 3.13  
- Можно убрать gil, но это замедляет однопоточные сценарии на 10% из-за изменения механизма подсчета ссылок  
- Jit - компилятор (ускоряет выполнение кода, т.к. частые участки кода преобразуются в машинный код во время выполнения)
