.keys() возвращает объект dict_keys([key1, key2]) - он является представлением словаря, поддерживает многократную итерацию и динамически изменяется с изменением словаря, поддерживает операции со множествами (объединения и пересечения).  
.values() - объект представления списка значений, не поддерживает операции со множествами  
.items() - объект представления списка кортежей с парами ключ значение, поддерживает операции со множествами  
.get(key, default=None) - возвращает значение ключа. если его нет, то возвращает значение по умолчанию  
dict.pop(key, default=None) - удаляет ключ и возвращает значение. исключений не бросает  
dict.popitem() - удаляет последнюю добавленную пару ключ значение и возвращает их в виде кортежа  
dict.setdefault(key, default=None) - возвращает значение для указанного ключа. Если ключ отсутствует, устанавливает его с значением default и возвращает default  
dict.update([other]) - обновляет словарь, добавляя пары ключ-значение из other. Если ключ уже существует, его значение будет перезаписано.  
.clear() - очищает словарь
