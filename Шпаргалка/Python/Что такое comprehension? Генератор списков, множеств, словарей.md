Что такое comprehension? Генератор списков, множеств, словарей  
List comprehension (или "генератор списков") — это способ создания нового списка на основе существующих данных с использованием краткой, лаконичной записи. Этот синтаксис позволяет избежать использования циклов for и функций, таких как map или filter, для генерации новых списков.  
  
Синтаксис:  
  
```  
[выражение for элемент in итерация if условие]  
```  
  
Работает быстрее циклов, потому что обработка выполняется на уровне С  
  
Генератор множеств:  
  
```  
{выражение for элемент in итерация if условие}  
```  
  
Генератор словарей:  
Cинтаксис:  
  
```  
{ключ:значение for выражение in итерация if условие}  
```  
  
Пример:  
  
```  
tuples_list = [('a', 1), ('b', 2), ('c', 3)]  
dict_comprehension = {key: value for key, value in tuples_list}  
```  
  
Генератор кортежей:  
Компрехеншна для кортежей нет, но можно создать генераторное выражение (не список!), а потом сделать его кортежем с помощью tuple().  
Пример:  
```  
tuple(x**2 for x in range(5))  
```
