HTTP (HyperText Transfer Protocol) версии 1.x — это протокол прикладного уровня, используемый для передачи данных в веб-приложениях. Он основан на клиент-серверной архитектуре, где клиент (обычно веб-браузер) отправляет запросы, а сервер отвечает на них.  
  
Запрос (Request):  
Запрос HTTP состоит из трех основных частей:  
1. Стартовая строка ответа (Status Line): состоит из метода, url и версии http  
2. Заголовки запроса (Headers): предоставляют дополнительную информацию о запросе или о клиенте.  
  
Пример:  
Host: www.example.com  
User-Agent: Mozilla/5.0  
Accept: text/html,application/xhtml+xml  
  
3. Тело запроса (Body): содержит данные, отправляемые клиентом серверу (например, при POST-запросах).  
Пример: JSON-данные, формы, XML.  
  
Ответ (Response):  
Ответ HTTP состоит из трех основных частей:  
4. Стартовая строка ответа (Status Line):  
Формат: HTTP/VERSION STATUS_CODE REASON_PHRASE  
Пример: HTTP/1.1 200 OK  
  
5. Заголовки ответа (Headers):  
6. Тело ответа (Body)
