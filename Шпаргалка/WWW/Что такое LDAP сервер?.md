Lightweight Directory Access Protocol сервер - специализированный сервер, использующий протокол LDAP для централизованного управления аутентификацией и авторизацией в корпоративных средах, обеспечивая единый источник истины для данных о пользователях. По структуре данных напоминает файловую систему (древовидную). Пример - OpenLdap.
