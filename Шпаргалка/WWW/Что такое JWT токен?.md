Json Web Token - стандарт передачи информации в виде компактного и автономного токена. Используется для аутентификации и авторизации.  
  
Компоненты:  
* Header - содержит инфу о типе токена и алгоритме шифрования  
* Payload - тело запроса с передаваемой информацией и claims (инфа по типу времени жизни токена, издателя и тд.)  
* Signature - подпись, которая создаётся путём шифрования Header и Payload  
  
Вид токена:  
* Строка вида Header.Payload.Signature, закодированная в base64  
  
Достоинства JWT токенов:  
* Компактность - токен небольшой по размеру, чем удобен для передачи по http  
* Автономность - Содержит в себе всю инфу для аутентификации, чем снижает количество обращений к внешним сервисам.  
* Безопасность - Подпись гарантирует неизменяемость токена злоумышленником
