cat - Выводит содержимое одного или нескольких файлов на стандартный вывод, а также объединяет файлы.  
tar - Архиватор, который используется для упаковки (архивирования) и распаковки файлов и каталогов в один архивный файл.  
awk - Мощный текстовый процессор и язык для анализа данных, позволяющий фильтровать, форматировать и обрабатывать строки файлов.  
sed - Потоковый редактор, применяемый для автоматизированного поиска, замены и преобразования текста в файлах или потоках данных.  
top - Утилита для мониторинга процессов и ресурсов системы в режиме реального времени (показывает загрузку CPU, память, список процессов и т.д.).  
man - Просмотр справочных страниц (мануалов) для различных команд и программ, позволяющий быстро получить информацию об их использовании.  
pwd - Выводит абсолютный путь текущего рабочего каталога.  
cut - Позволяет вырезать определённые части (столбцы или поля) строк из текстовых файлов или входного потока данных.  
ssh - Клиент для безопасного удалённого доступа по протоколу Secure Shell, обеспечивающий шифрованное соединение с удалёнными системами.  
yes - Выводит заданную строку (по умолчанию — «y») бесконечно, что может быть полезно для автоматического подтверждения при работе с другими командами.  
git - git  
vim - текстовый редактор
