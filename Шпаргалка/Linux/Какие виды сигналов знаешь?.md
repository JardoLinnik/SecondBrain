Отправка сигнала происходит командой kill *pid*. по умолчанию отправляется SIGTERM, но можно указать другой, добавив ключ с названием или номером сигнала.  
  
1. Сигналы, вызывающие завершение процесса  
* SIGINT (номер 2)  
- Действие по умолчанию: Завершение процесса  
- Можно перехватить: Да  
- Описание: Прерывание процесса (обычно отправляется при нажатии Ctrl+C).  
  
* SIGTERM (номер 15)  
- Действие по умолчанию: Завершение процесса  
- Можно перехватить: Да  
- Описание: Запрос на завершение процесса (команда kill по умолчанию).  
  
* SIGHUP (номер 1)  
- Действие по умолчанию: Завершение процесса  
- Можно перехватить: Да  
- Описание: Сигнал о разрыве соединения с терминалом или перезагрузке конфигурации.  
  
* SIGQUIT (номер 3)  
- Действие по умолчанию: Завершение с дампом памяти  
- Можно перехватить: Да  
- Описание: Завершение процесса с созданием дампа памяти (обычно отправляется при нажатии Ctrl+).  
  
* SIGUSR1 (номер 10)  
- Действие по умолчанию: Завершение процесса  
- Можно перехватить: Да  
- Описание: Пользовательский сигнал 1 для специальных нужд приложений.  
  
* SIGUSR2 (номер 12)  
- Действие по умолчанию: Завершение процесса  
- Можно перехватить: Да  
- Описание: Пользовательский сигнал 2 для специальных нужд приложений.  
  
* SIGALRM (номер 14)  
- Действие по умолчанию: Завершение процесса  
- Можно перехватить: Да  
- Описание: Сигнал таймера, отправляется по истечении установленного времени.  
  
* SIGKILL (номер 9)  
- Действие по умолчанию: Немедленное завершение процесса  
- Можно перехватить: Нет  
- Описание: Принудительное завершение процесса.  
  
2. Сигналы управления выполнением процесса  
* SIGSTOP (номер 19)  
- Действие по умолчанию: Остановка процесса  
- Можно перехватить: Нет  
- Описание: Приостанавливает процесс.  
  
* SIGCONT (номер 18)  
- Действие по умолчанию: Продолжение процесса  
- Можно перехватить: Нет  
- Описание: Возобновляет выполнение приостановленного процесса.  
  
3. Сигнал для уведомления родительского процесса  
* SIGCHLD (номер 17)  
- Действие по умолчанию: Игнорирование  
- Можно перехватить: Да  
- Описание: Отправляется родительскому процессу при завершении дочернего процесса.
