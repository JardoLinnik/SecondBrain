Для асинхронного чтения файла можно использовать библиотеку aiofiles  
Которая предоставляет асинхронный аналог стандартного чтения файла.  
  
import asyncio  
import aiofiles  
  
async def read_file_async(f_path):  
async with aiofiles.open(f_path, mode='r') as f:  
content = await f.read()  
  
asyncio.run(read_file_async('example.txt'))  
  
Если не использовать библиотеку, то нужно открывать файл в отдельном потоке, чтобы обеспечить совместимость асинхронного кода с синхронным IO-вызовом
