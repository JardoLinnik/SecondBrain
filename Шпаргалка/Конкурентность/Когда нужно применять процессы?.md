Когда использовать:  
* CPU-интенсивные задачи: При выполнении CPU-bound операций, которые могут быть распараллелены по ядрам процессора.  
* Необходимость изоляции: Когда требуется полная изоляция между задачами для обеспечения безопасности или стабильности.  
* Обход ограничений GIL: Для выполнения параллельных вычислений в CPython, где GIL ограничивает многопоточность.
