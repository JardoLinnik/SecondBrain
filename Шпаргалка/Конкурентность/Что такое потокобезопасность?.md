Потокобезопасность - это концепия, которая описывает способность программы или её компонентов правильно работать в многопоточном окружении.  
  
### Основные аспекты потокобезопасности:  
В потокобезопасной программе данные, к которым обращаются несколько потоков, защищены от конкуретного доступа. Реализуется с помощью элементов синхронизации ( мьютексы, семафоры и др)  
  
Это позволяет избежать race condition, где два или более потоков пытаются одновременно изменить одни и те же данные  
  
from threading import Thread, Lock  
  
counter = 0  
lock = Lock()  
  
def increment():  
global counter  
for _ in range(1_000_000):  
with lock:  
counter += 1  
  
t1 = Thread(target=increment)  
t2 = Thread(target=increment)  
t1.start()  
t2.start()  
t1.join()  
t2.join()  
print(counter)  
  
Пример показывает, как защитить объектом синхронизацию участок, в котором будет race condition в случае его отсутствия
