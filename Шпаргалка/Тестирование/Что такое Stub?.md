Stub (заглушка) — это простейшая версия объекта или функции, которая используется в тестировании для имитации поведения реальных компонентов. Stub возвращает предопределённые значения вместо выполнения реального кода.  
  
По сути это обычный класс или функция с имитацией поведения оригинала.  
  
Пример:  
  
```  
def stubbed_api_call():  
return {"status": "success", "data": [1, 2, 3]}  
```
