Helm Chart — это пакетный формат в Kubernetes, используемый для управления, развертывания и обновления приложений. Он содержит все файлы и конфигурации, необходимые для развертывания Kubernetes-приложения.  
Helm — это инструмент для управления Kubernetes-приложениями с использованием Charts, которые представляют собой набор шаблонов Kubernetes-манифестов (YAML-файлов).  
  
Почему использовать Helm?  
* Упрощённое управление: Объединяет сложные Kubernetes-манифесты в один пакет.  
* Повторяемость: Позволяет развернуть одно приложение в разных средах с разными настройками.  
* Обновление приложений: Обеспечивает лёгкое обновление или откат приложений.  
* Сообщество: Существует множество готовых Helm Charts, которые можно использовать.  
  
Helm Chart — это структура каталогов с определёнными файлами:  
mychart/  
Chart.yaml # Основная информация о пакете  
values.yaml # Значения по умолчанию  
templates/ # Шаблоны Kubernetes-манифестов  
charts/ # Зависимости
