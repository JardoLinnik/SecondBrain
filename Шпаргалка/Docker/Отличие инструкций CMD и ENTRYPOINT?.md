Работают одинаково, если не использовать аргументы при запуске и не использовать их вместе.  
CMD можно полностью переопределить при подаче аргументов при запуске контейнера:  
  
CMD ["echo", "Hello, world!"]  
  
Если мы напишем:  
docker run <image> echo "Hello, Docker!"  
  
То выведется "Hello, Docker!" и инструкция CMD будет полностью переопределена.  
  
ENTRYPOINT нельзя переопределить, можно только дополнить аргументами.  
  
ENTRYPOINT ["echo", "Hello,"]  
  
CMD ["world!"]  
  
Если мы напишем:  
  
docker run <image> "Docker!"  
  
то будет выведено "Hello, Docker"
