Docker - это платформа для разработки, доставки, запуска приложений в изолированных средах (контейнерах), которая помогает разработчикам создавать и обслуживать их приложения в более простой и организованной манере. Можно поместить своё приложение и инструкции по его запуску в контейнер и запустить его на любой другой машине простыми командами.  
  
Для чего нужен Docker?  
  
1. **Изоляция**: Можно запустить контейнеры изолированно друг от друга  
  
2. **Удобство**: Контейнер можно запустить одной командой, т.к. все инструкции по запуску находятся внутри  
  
3. **Одинаковая работа везде**: Можно легко запускать контейнер на разных машинах
