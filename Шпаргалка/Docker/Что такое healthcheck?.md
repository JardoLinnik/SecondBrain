Healthcheck в Docker — это механизм, позволяющий контейнеру «самопроверяться» на корректность работы. С помощью директивы HEALTHCHECK в Dockerfile (или соответствующих настроек в Docker Compose) вы можете указать команду, которая регулярно проверяет состояние приложения внутри контейнера. На основании результатов проверки Docker присваивает контейнеру статус healthy или unhealthy.
