1. Создать Dockerfile, в котором вы опишете, какой базовый образ используете, какие зависимости устанавливаете, какие файлы копируете и какую команду запускаете внутри контейнера.  
2. Выполнить команду docker build, чтобы на базе Dockerfile собрать образ.  
3. Выполнить команду docker run, чтобы создать и запустить контейнер из собранного образа.
