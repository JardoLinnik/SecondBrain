Docker Registry — это сервис (или хранилище), в котором хранятся образы Docker. Он позволяет публиковать образы (push) и скачивать их (pull) в/из различных окружений. Проще говоря, это репозиторий для контейнерных образов, аналогичный тому, как Git-репозитории служат для кода.  
  
Основные Registry: Docker Hub, Amazon/Google/Nexus/Gitlab/Github/Azure Container Registry
