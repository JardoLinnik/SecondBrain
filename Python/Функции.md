Функции - это именованный фрагмент кода который можно вызвать несколько раз и добавлять функциональность и логику коду. Функцию можно вызвать и определить. Для определения фнкции используется ключевое слово def.
`def name_function (parametr1, parametr2):
	`pass`

Функции тоже являются обьектом и могут использоваться как аргументы других функций или в циклах.

### Аргументы 

Аргументы - знчения которые встают за место зданных прараметров.

Позиционные аргументы - аргументы присваиваются параметрам согласно порядку.

Аргументы ключевые слова - присваиваютя параметрам путем указания на эти параметры при вызове. Есть возможность присвоить значение аргумента при опре делении функции.
Значения по умолчанию высчитываются при определеении функции, а не при вызове, что нужно учесть при попытке по умолчанию задать изменяймую структуру.

`*args` - будет выведено как кортеж с набором элементов. Указывается в качестве параметра и использует все элементы после включая их в кортеж. Стоит на третьем позиционном месте после позиционных и ключевых аргументов. 

`**kwargs`- будет выведено как список. Входят аргументы со значениями и создается список.

### Docstring
Cтрока документации - создается после задания функции. Описывается функционал и оставляется подскказка для поддержки кода. При вызове метода help(function_name) отображает строку документации. Или можно использовать встроенный метод `__doc__`

### Внутренние функции



### pass
Ключевое слово pass. Используется для закупоривания функции когдла она не имеет функционала и временно является пустой.

### return
Ключевое слово возвращающе значения и звершающее функцию. 

### None
Отсутствующее значение. Не является пустым значением и не == False.

### Встроенные функции
Sum() - сумма обьектов итерабельного обьекта.
