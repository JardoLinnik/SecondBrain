# 1. Python : с чем его едят?

Переменная - имя значения которое мы хотим использовать в программе. Ящик для хранения данных.

Статический язык - это язык в котором типы переменных указываются заранее и не могут быть изменены. Обычно имеют свойство компилироваться.

Компиляция - преобразование текущего кода в более низкоуровневый (машинный код) в виде отдельного файла  для последующего использования машиной.

Динамический язык (скриптовый язык) - противоположность статическому, переменные и типы не надо указывать изначально. Обычно интерпритируются. Более медленные чем статические, но более простые.

Интерпритация - динамическое преобразование кода в машинный или промежуточный код и построчное выполнение в реальном времени.
\
Сценарий - короткая программа обычно написанная на динамическом языке.

Связующий код - сценарий для подготовки данных обработке более длинным статическим кодом.

Python быстрый язык, но не достаточно быстрый для всех типов программ. Если программа занимается исключительно вычислениями "ограничена быстродействием процессора" (CPU bound) то лучше использовать C, C++ или JAVA. Однако не всегда :

- Лаконичное и быстрое написание кода Python дает больше запас времени на написание более качественного алгоритма решения, что может превзойти скорость работы менее качественного алгоритма, но на более низкоуровневом языке. 

- Приложения в которых сервер может долго отвечать и скорость расчета процессора будет одинакова как на динамических так и на статических языках.

- Оптимизация кода с помощью дополнительных вставок кода написанного на C (на котором написан интерпретатор Python).

- Имеются  и такие случаи когда Python будет не вывозить нагрузку в таких случаях придется использовать другой язык. Отличной альтернативой является GO - высокоуровневый язык похожий на Python, но имеющий более высокую производительность.

# 2. Ингредиенты Python : числа, строки и переменные.

В Python все является объектом: типы, переменные, функции, программы и др. Это помогает Python быть стабильным языком. Объект похож на коробку с данными. Объект имеет свой тип данных для определения с какими объектами он может взаимодействовать и каким образом. Тип также определяет можно ли поменять значение. 

Типизируемость - свойства языка менять типы. Python является строго типизируемым языком, а значит он не может менять тип хотя может менять значение.

Переменные ссылаются на ячейки в памяти компьютера и являются именами ячейки на которую ссылаются.
