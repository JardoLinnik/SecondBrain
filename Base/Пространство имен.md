Пространство имен это внутенее разделеенияе памяти под обьекты. 

Виды:
	Built-in - внутри python
	Global -  внутри модуля
	Enclosing - переменные внешней функции
	Local - переменные внутри функции

- У классов свое пространство имен. Внутри методов недоступны переменные внутри класса. Если не привязать self, cls, staticmethod, classmethod.
