Конструкция для читаймости кода, которая делает что-то до действия и после. Дейтвием может служить открытие соединения.

- Обрабатывает исключения в __exit__
- Прописать автозакрытие подключений
- Ассинхронный контекстный менеджер


Применение: Подключение к чему-то


**Пример**:
```
from contextlib import contextmanager

def open_resource(*args):
	resource = None
	try:
		resource = Resource()
		resource.open(*args)
		yield resource
	except:
		#log
		raise
	finally:
		if resource:
			resource.close()
			
			

class ContextManager:
	def __init__(self):
		self.args = args
		self.resource = None
	
	def __enter__(self):
		self.resource = Resource()
		self.resource.open(*self.args)
		return self.resource
		
	def __exit__(self, exc_type, exc_val, exc_tb):
		if self.resource:
			self.resource.close()
		
```
