[[Пакетный менеджер]] для работы c Python. Является более проработанной версией стандартного пакета [[pip]]  

**Команды**:
- `poetry init` - инициализирует poetry в директории
- `poetry new {name}` - создает шаблонный poetry проект
- `poetry add {pocket}` - добавляет пакет в poetry фаил 
- `poetry shell` - активирует [[*Виртуальное окружение]].
- `poetry install` - устанавливет пакеты в виртуальное окружение.
- `poetry update` - обновляет пакеты.
- `poetry build` - 
- `poetry run` - запуск чего то отдельного в виртуальном окружении
- `poetry show --tree` - показывает все зависимости пакетов.
- `poetry show --latest` - показывает последние версии наших пакетов.

**Проблема**:  для более строгого и современного управления зависимостями и проектами.
