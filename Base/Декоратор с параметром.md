Декоратор которы использует дополнительные параметры.

- Для такого декоратора нужна будет еще одна обертка так как параметры идут в сам декоратор а внутрення функция wrapper заменяет декоратор на принятие функции.
- Нельзя забывать что параметр должен быть nonlocal для того чтобы его было видно. 


Применение: Декораторы с каким то параметром.

**Пример**:
- Лимитиованный декоратор
```
def deco(limit):
	def wrapper(func):
		def inner(*args, **kwargs):
			nonlocal limit
			if limmit = 0:
				print("Нельзя вызвать функцию")
			
			res = func(*args, **kwargs)
			limit -= 1
			return res
		return inner
	return wrapper
```
