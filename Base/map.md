Используется для использование функции над итерируемым объектом и возвращения измененной последовательности.

Применение: Когда нкжно задать последовательность с определенной обработкой.

**Пример**:

```
nums = list(map(int, input().split()))
#Обработка введенный цифр и ввод их в список.


list1 = [1, 2, 3]
list2 = [4, 5, 6]
sum_list = list(map(lambda x, y: x + y, list1, list2))
# Результат: [5, 7, 9] - обработка нескольких списков

```
