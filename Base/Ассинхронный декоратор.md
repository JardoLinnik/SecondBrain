Декоратор для ассинхронной функции.

Применение: Для ассинхронной функции.


**Пример**:
```
def deco(coroutine: Coroutine):
	async def wrapper (*args, **kwargs):
		res = await coroutine(*args,**kwargs)
		return res
	return wrapper

@deco
async def my_async_func():
	return 1

```
