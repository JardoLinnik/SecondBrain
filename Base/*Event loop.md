Event Loop(цикл событий) - механизм в ассинхронном програмировании, позволяющий эфективно обрабатывать множество задач, не создавая большое количество потоков.

Event loop - Непрерывный цикл ожидающий наступление событий(завершение операции ввода-вывода, приход запроса или ответа, таймеров)

- Когда событие приходит оно помещается в очередь задач
- Event loop - поочередно вытаскивает задачи из очереди не блокируя основной поток, тоесть выполняя задачи ассинхронно и неблокирующе
- После выполнения Event loop входит в режим ожидания и ждет следующих задач.
