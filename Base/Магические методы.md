Магические методы — это специальные методы с двумя подчёркиваниями, с помощью которых объект реагирует на операторы, функции и синтаксические конструкции Python.


`__init__` - инициализация обьекта
`__new__` - создание обьекта

`__str__` - отобржение для вывода
`__repr__` - отображение обьекта для разработчика

`__abs__` - поведение при вызове abs(модуля)
`__len__` - поведение при вызове len(длинна)

`__add__` - поведение при сложении 
`__radd__` - меняем стороны также и для остальных
`__mul__` - поведение при умножении
`__sub__` - поведение при вычитании
`__truediv__` - поведение при делении

`__eq__` - поведение при ==
`__ne__` - поведение при !=
`__lt__` - поведение при <
`__le__` - поведение при <=
`__gt__` - поведение при >
`__ge__` - поведение при >=

`__hash__` - слетает когда мы используем `__eq__`. Позволяет хешировать обьект.

`__bool__` - задает локику отношения числа к определенному bool значению. Если нет `__bool__`, вызывается `__len__` и смотрит значение. Если нет ничего то всегда True.

`__call__` - делает обьект похожим на функцию, но которая хранит состояние.

`__getitem__` - для получения значения по ключу или индексу
`__setitem__` - для изменения по индексу или ключу
`__delitem__` - для удаления по индексу или ключу


`__iter__` - делает обьект итерируемым
`__next__` - возвращает следующий элемент при итерации
