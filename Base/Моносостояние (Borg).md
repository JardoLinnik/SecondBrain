Прием когда несколько обьектов имеют общее состояние, даже если сами обьекты - разные экземпляры.

- Альтернатива Singleton, однако обьектов много и делят состояние


**Применение**:

1. Конфиги
2. Логирование
3. Кеширование
4. Пул подключения или общие параметры подключений

**Пример**:
```
class Borg:
    _shared_state = {}   # общее состояние для всех объектов

    def __init__(self):
        self.__dict__ = self._shared_state


```
