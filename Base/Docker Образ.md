Шаблон для создания контейнеров. Образ это та среда которую мы хотим перевезти на другое устройство и запускать код.

- Фиксированые и неименяймые. Взаимодействие только через чтение.
- Цепочка наследований Image
- Scratch
	- Образ может быть даже пустым, но это усложнит работу поскольку для отправки на другое устройство придется постоянно писать код в контейнер
- Что содержит
	- Исходный код
	- Библиотки и зависимости
	- Настройки и инструкции
