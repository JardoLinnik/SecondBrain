
- open('file_name',mode='', encoding ='')
	- r - чтение
	- w - запись, содержимое файла удаляется, а если файла нет он создается
	- x - создание нового файла для записи
	- a - запись без удаления содержимого файла
	- b - открытие в бинарном режиме
	- t - открытие в текстовом режиме
	- + - открытие файла для чтения и записи
- Не забывать о закрытии файла
	- f.close
- Записать в файл
	- str = ''
	- f = open(file.txt, mode = 'a', encoding = 'utf-8')
	- f.write(str)
	- f.close()

Применение: небольшое открытие и закрытие файла для остального лучше подходит менеджнр контекта
