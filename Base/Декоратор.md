Структурный паттерн проектирования необходимый для изменения работы функции не меняя ее структуру, а оборачивая ее в обертку.


- Вызов декоратора без сахара
	- func = deco(func)

- [[Декоратор с параметром]]
- [[Ассинхронный декоратор]]
- [[Мемоизация]]
- [[Контекстный менеджер]]

- Встроенные декораторы
	- classmethod
	- staticmethod
	- property

- Сохранение метаданных
	- [[functools.wraps(func)]]

Применение: когда нужно добавить новый контент в обьект

- Для тестирования
- Для валидации
- Для работы с api

**Пример**:

- Подсчет времени выполнения программы
```
import time

def time_deco(func: Callable):
	def wrapper(*args,**kwargs):
		start = time.time()
		res = func()
		end = time.time()
		print(f"Исполнение заняло {end - start}")
		return res
	return wrapper
```
- Логирование параметров функции
- Повтор функции в случае ошибки
- Ограничение на частый вызов функции


```
def deco (func):
	fuctools.wraps(func)
	def wrapper(*args, **kwargs)
		print(func.__name__)
		return func(*args, **kwargs)
	return wrapper

@deco	
def magic(x):
	return x
	
magic(24)
```
