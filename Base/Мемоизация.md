Процес сохранения промежуточных результатов выполнения программы(функции), для предотвращения повторного вычисления. Способ оптимизации программы.

- Нужно помнить о памяти так как она забивается.

Применение:Для ускорения работы кода. 

**Пример**:
```
from functools import lru_cache
import time

@lru_cache(maxsize=5)
def my_long_calc():
    time.sleep(3)
    return 42

print(my_long_calc())
print(my_long_calc())
print(my_long_calc())

```
