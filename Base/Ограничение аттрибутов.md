`__slots__` - специальный механизм Python ограничивающий набор аттрибутов обьекта, делающий обьекты легче.


# Сравнение: обычный класс vs класс со slots

### Без slots:

- Каждый объект имеет `__dict__`
- Можно добавлять любые атрибуты
- Занимает больше памяти
### Со slots:

- Нет словаря → только фиксированные поля
- Объекты «тоньше» и быстрее
- Идеально для миллионов однотипных объектов


### ✔ Высоконагруженные бэкенды
Например, в сервисах с большим количеством инстансов моделей, где оптимизируют каждое действие.
### ✔ Парсеры / обработка данных
Когда создают тысячи объектов одной структуры.
### ✔ Игровые серверы
Где важно, чтобы объекты были лёгкими.
### ✔ Data science (Python-level оптимизация)
До применения numpy/pandas.


**Применение**:
Используется в классов **только тогда**, когда нужно много маленьких объектов → экономия памяти может стать огромной.
