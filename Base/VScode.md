Легкий редактор кодда сосредоточенный на расширяймости и простое. Поддерживает множество языков програмированния. Подходит для крупных проектов.


- Команды:
	- alt  - -курсор в нескольких местах
	- alt + стрелки -- перемещает строку кода
	- alt + shift -- выделяет строки для одновременного написания
	- ctrl + H    -- поиск и замена 
	- ctrl + / -- коментирование
	- ctrl + G -- перейти к строке
	- ctrl + F -- поиск
	- ctrl + Z -- отменить действие
	- ctrl + Y -- вернутся к отмененному
	- F12 -- пееносит к обьявлению выделенного обьекта
