Счетчик - встроенная конструкция в Python collection.

Функция collections.Counter() принимает на вход последовательность и выдает словарь с ключами как элемент и количесвом его в последовательности.

- Метод `most_common(n)` ищет `n` самых повторяющихся элементов. Найдём для примера три наиболее частых символов. 
```
letter_cnt.most_common(3) [('а', 5), ('б', 2), ('р', 2)]
```

- Нередко интерес представляют не самые частотные, а уникальные значения, самые редкие элементы. Их можно найти срезом с шагом `-1`:

- Счетчик в сочетании с [регулярными выражениями](https://proglib.io/p/regex-for-beginners/) используется для [частотного анализа текста](http://nlpx.net/archives/29).

Применение: Подсчет элементов последовательности.

**Пример**:

```
list_of_letters = list('абракадабра')
letter_cnt = collections.Counter(list_of_letters)
letter_cnt
->
Counter({'а': 5, 'б': 2, 'р': 2, 'к': 1, 'д': 1})
```
