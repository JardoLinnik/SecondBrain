Приложение нагруженное данными. I/O bound приложения.

Таким приложениям нужно :
- Хранить данные, чтоб эти или другие приложения могли найти их в дальнейшем ([[База данных]])
- Запоминать результаты ресурсоемких операций для ускорения чтения ([[Кеширование]])
- Предоставлять возможность поиска данных по ключевым словам или фильтровать данные([[Поисковые индексы]])
- Отправлять сообщения другим процессам для ассинхронной обработки([[Потоковая обработка]])
- Время от времени перемалывать большие обьемы данных ([[Пакетная обработка]])
