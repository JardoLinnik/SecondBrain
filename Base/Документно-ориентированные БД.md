Это база данных направленная на хранение и запрос документов по типу [[JSON]].

Подходит для задач где данные неструктурированные. Хранение какого то одного документа не связанного.

![[ru_document_oriented_databases_2.webp]]

- Особенности
	- Отсутствие схемы данных
	- Документе в базе адресуются с помощью уникального ключа. Для ссылки на запись или документ.
	- Разная реализация в каждой отдельной базе.
	- Не подходит для отношений баз данных.
