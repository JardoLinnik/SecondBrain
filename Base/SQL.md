- **Команды**:
	- `SELECT` - Получение выборок данных
		- `SELECT * FROM [TABLE_NAME]` - вывод всех данных таблицы
		- `SELECT id, member_name AS Name FROM table_1` - Псевдоним 
	- `DESCRIBE` - для уточнения типов данных атрибутов.
	- `DISTINCT ` - исключение дубликатов
	- `WHERE` - условный оператор
		- Задает условия выборки
			- `SELECT * FROM Student WHERE first_name = "Grigorij" AND YEAR(birthday) > 2000;
		- Логические операторы  
			- Сначала — NOT, Затем — AND, Потом — XOR, В конце — OR
		- Дополнительные операции
			- IS NULL - проверяет значение переменной пустое ли оно.
			- BETWEEN  min AND max  - проверяет значения между интервалами
			- IN (" "," " ) - проверяет входит ли в список значений.
		- `LIKE` - оператор сравнения
			- `SELECT name, email FROM Users WHERE email LIKE '%@hotmail.%'
		- `REGEXP`  -  более усложненная система поиска.
	- `ORDER BY` - сортировка данных
	- `GROUP BY` - групировка 
		- Агрегатные функции - функции производящие вычисление на наборе значений и выдающие одно		
			-  SUM
			- COUNT
			- AVG
			- MIN
			- MAX
	- `HAVING` - условный оператор для групп
	- `JOIN`
		- `INNER` - внутренее соединение. При котором пары записей из двух таблиц соединяются в промежуточную таблицу.
		- `OUTER` - внешнее соединение
			- `LEFT
			- `RIGHT
			- `FULL
	- `LIMIT` - Задает диапозон записей таблиц.
		- `OFFSET` - Сколько записей от начала пропустить.
