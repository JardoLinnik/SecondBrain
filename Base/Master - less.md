Нет ведущих узлов

- Пишем в определненные ноды 
- Читаем с определенных нод 

W - реплики для записи
R - реплики для чтения
N - сумарное кол-во репоик

W + R > N - Строгая согласованность
W + R <= N - Не гарантирует строгой согласовоности
R = 1 & W = N - Быстрое чтение
W = 1 & R = N - Быстрая запись

**Применение**:
Модель P2P
Нет централизованных узлов.
