.isdigit() - проверить есть ли числа.

### Списки list [ ]

Списки - последовательность элементов с возможностью эту последовательность менять и проводить над ней маннипуляции. Для уникальных значений лучше подойдет set.

```
#Пустой список
x = [ ]
list()
```

```
x = [0,2,3,4,5,6,1,2]
x[0] = 0
```

Метод .append() - добавление элемента в конец списка.

Метод .extend() - объединение списков. Также можно просто написать +=

Метод .insert() - добавление элемента в определенную позицию.

del - удаление позиции. Не метод.

Метод.remove() - удаление значения.

Метод.pop() - получение элемента и последующее удаление.

LIFO - append() добавляет элемент в конец списка, pop() удаляет этот элемент - стек.
FIFO - append() добавляет элемент в конец списка, pop(0) удаляет элемент в начале - очередь.

Метод.index() - узнать позицию указанного элемента.

in - оператор проверки элемента в списке.

Метод.count() - проверка дубликатов и вывод кол-ва

sorted.x - отсортированная копия 
.sort - метод сортирующий сам список.

len() - получение длинны списка.

### Кортежи tuple( )

Не изменяемые последовательности.
x = ()

Распаковка кортежа - присваивание нескольким элементам позиций.
a, b, c = tuple

- Кортежи занимают меньше места.
- уменьшение вероятности ошибки
- кортежи как ключи словаря.
- именованные кортежи.
- аргументы функции передаются как кортежи.
